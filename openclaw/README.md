# OpenClaw Identity Files

These files define Clawster's personality for OpenClaw.

## Setup

Copy these files to your OpenClaw workspace:

```bash
cp IDENTITY.md SOUL.md ~/.openclaw/workspace/
```

Then restart the OpenClaw gateway:

```bash
openclaw gateway restart
```

## Files

- **IDENTITY.md** - Who Clawster is (name, appearance, capabilities)
- **SOUL.md** - How Clawster behaves (personality, communication style)
